🔧 嵌入式知识：PWM 生成模拟量与低通滤波

PWM 加低通可表达平均模拟量，但不是天然精密 DAC

真实输出还受 GPIO 驱动电阻、高低电平压降、电源噪声、负载、漏电和滤波器影响。若后级输入阻抗与 RC 电阻同量级，会改变直流增益；驱动低阻负载通常需要缓冲器。PWM 的基准实际上是 I/O 电源，因此电源漂移会直接成为增益误差。

边沿对齐计数器的理想 duty 档位由 timer period 决定；频率大致为 f_timer / ((PSC+1)*(ARR+1))，具体端点和中心对齐模式必须按定时器手册确认。提高 PWM 频率便于滤除载波，却在固定 timer clock 下减少计数档位；增加位数则降低载波频率。TI 的 PWM-DAC 资料也把 duty resolution、载波谐波/纹波和滤波器带宽视为共同误差预算，而不是独立参数。

低通在纹波、带宽和建立时间之间取舍

一阶 RC 的截止频率 f_c = 1/(2πRC)。令 f_c 远低于 PWM carrier 可减少纹波，但目标电压变化会更慢；提高滤波器阶数能改善载波抑制，却增加器件误差、成本和可能的稳定性问题。必须从允许纹波、信号带宽、最大建立时间和负载共同求解，不能套用固定的“载波十倍”经验。

duty 更新应与 timer update event 同步并使用 preload，避免周期中途改 compare 产生 runt pulse。DMA 波形还要处理缓冲切换、欠载和更新率。0%/100% 是否真正恒低/恒高、互补输出死区是否介入、timer clock 倍频规则均为芯片特性。

适用边界

PWM+LPF 适合慢速设定值、偏置、背光和成本敏感的波形。它不适合要求低噪声、快速建立、高绝对精度、单调性认证或四象限输出的场合；此时应评估片上/外部 DAC。若输出进入闭环，滤波器相位和更新延迟也必须进入控制模型。

常见误区

  • 8 位 duty 就声称有 8 位有效精度，忽略纹波、电源、非线性和噪声。
  • 空载示波器波形合格就认为连接真实负载后不变。
  • 只降低 RC 截止频率消除纹波，不检查建立时间。
  • 在周期中途直接写 compare，产生额外窄脉冲。
  • 用万用表平均值验证动态波形和频谱。

验证清单

  • 记录 timer source、PSC/ARR、对齐模式、实际 carrier、duty 档位和更新同步点。
  • 在 0%、最小非零、50%、最大非满和 100% duty 下测 DC、纹波、单调性与负载误差。
  • 对最坏电源、温度、器差和负载计算/仿真滤波器，再测 step settling 与频谱。
  • 注入连续 duty 更新和 DMA 欠载，捕获 runt pulse、毛刺与缓冲边界。
  • 若进入 ADC/控制环，避免采样与 PWM 拍频，并验证相位延迟和 aliasing。

自测

把 RC 电容增大十倍一定能改善 PWM 模拟输出吗？它通常降低载波纹波，但也使建立时间变慢，并可能增加启动、漏电和驱动负担；必须按完整指标判断。
