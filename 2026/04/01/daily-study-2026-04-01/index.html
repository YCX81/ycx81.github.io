🔧 嵌入式知识：C/C++ 指针与内存模型

1. 指针的本质与大小

在嵌入式 C/C++ 中，指针不仅是变量地址的别名，更是直接操控硬件寄存器和内存映射外设的核心工具。

  • 指针变量存储的是地址值，大小取决于目标架构（32位 MCU = 4 字节）
  • void* 是通用指针，但不能直接解引用
  • 函数指针用于回调机制，是 HAL 驱动的灵魂

2. 指针算术与对齐

  • ptr + 1 移动 sizeof(*ptr) 字节，不是 1 字节！
  • ARM Cortex-M 要求 32 位访问 4 字节对齐，否则触发 HardFault
  • 结构体内有 padding 以满足成员对齐要求

3. volatile 指针 vs 指针 to volatile

  • volatile uint32_t *reg → 指向的内容是 volatile（硬件寄存器）
  • uint32_t * volatile ptr → 指针本身是 volatile（中断中可能改变）
  • 嵌入式最常用的是第一种：指向寄存器的 volatile 指针

4. C++ 智能指针在嵌入式的取舍

  • unique_ptr 零开销，可用于 RAII 管理外设资源
  • shared_ptr 有引用计数开销 + 动态分配，嵌入式慎用
  • 自定义 deleter 可包装硬件释放逻辑

5. 常见误区

误区 说明
sizeof(指针) == sizeof(int) 在 64 位主机编译时不成立
对 void* 做算术运算 GCC 允许但非标准，MISRA 禁止
忘记 restrict 关键字 编译器无法优化有别名的指针
返回栈上局部数组地址 函数返回后内存已无效

面试常见问法

Q: 什么是 dangling pointer？如何避免？
A: 指向已释放内存的指针。避免方法：释放后置 NULL、使用 RAII（C++）、静态分析工具检查生命周期。

Q: const int* 和 int* const 有什么区别？
A: const int* — 指向的值不可改；int* const — 指针本身不可改。记忆口诀：const 修饰它左边的东西（左边没有就修饰右边）。

Q: 嵌入式中为什么要用 volatile？
A: 告诉编译器该变量可能被外部因素（硬件、中断、DMA）改变，禁止优化掉对它的读写。

  1. polymorphism /ˌpɒlɪˈmɔːrfɪzəm/ — 多态性 技术

    Runtime polymorphism in C++ is implemented through virtual functions and vtables.

  2. inheritance /ɪnˈhɛrɪtəns/ — 继承 技术

    Use inheritance to share common peripheral driver interfaces across chip families.

  3. destructor /dɪˈstrʌktər/ — 析构函数 技术

    The destructor releases the DMA channel when the transfer object goes out of scope.

  4. constructor /kənˈstrʌktər/ — 构造函数 技术

    A constexpr constructor allows compile-time initialization of peripheral register maps.

  5. vtable /ˈviːteɪbəl/ — 虚函数表 技术

    Each class with virtual functions carries a hidden vtable pointer, adding 4 bytes on 32-bit targets.

  6. recursion /rɪˈkɜːrʒən/ — 递归 技术

    Avoid deep recursion on embedded targets where stack space is limited to a few kilobytes.

  7. variadic /ˌvɛriˈædɪk/ — 可变参数的 技术

    The printf family uses variadic arguments, which bypass type checking at compile time.

  8. constexpr /kɒnstˈɛkspɜːr/ — 常量表达式（C++编译期求值） 技术

    Mark baud rate divisor calculations as constexpr so the compiler resolves them at build time.

  9. nullptr /ˈnʌlpɪtər/ — 空指针（C++11） 技术

    Prefer nullptr over NULL to avoid ambiguity with integer overloads in C++.

  10. bitwise /ˈbɪtwaɪz/ — 按位的 技术

    Bitwise AND with a mask clears the unwanted bits in a hardware status register.

  11. arithmetic /əˈrɪθmətɪk/ — 算术的 技术

    Pointer arithmetic advances by the size of the pointed-to type, not by one byte.

    The FPU coprocessor must be enabled in the SystemInit function before any float operations.

    Always convert network packets to host endianness before parsing multi-byte fields.

  14. linkage /ˈlɪŋkɪdʒ/ — 链接属性 技术

    Declaring a function static gives it internal linkage, hiding it from other translation units.

  15. namespace /ˈneɪmspeɪs/ — 命名空间 技术

    Wrap driver APIs in a namespace to avoid symbol collisions in large firmware projects.

  16. template /ˈtɛmplɪt/ — 模板 技术

    A C++ template can generate type-safe ring buffers without runtime overhead.

  17. singleton /ˈsɪŋɡəltən/ — 单例 技术

    The UART driver is implemented as a singleton to prevent multiple instances accessing the same peripheral.

  18. instantiate /ɪnˈstænʃieɪt/ — 实例化 技术

    The linker will instantiate every template used, so monitor code size on constrained targets.

  19. overload /ˌoʊvərˈloʊd/ — 重载 技术

💬 地道表达

1. “hit the nail on the head”

意思： 一针见血，说得完全正确
场景： 当同事精准找出问题根因时使用

A: I think the bug is caused by struct padding mismatch between the host and the MCU.
B: You hit the nail on the head — the host sends 12 bytes but the MCU expects 8.

引用：「You hit the nail on the head.」— To Kill a Mockingbird, 1962

2. “a blessing in disguise”

意思： 塞翁失马，因祸得福
场景： 描述看似坏事最终带来好结果的转折

A: The chip shortage forced us to redesign the whole board.
B: Turned out to be a blessing in disguise — the new design uses half the power.

引用：「It was a blessing in disguise.」— The Shawshank Redemption, 1994

3. “bite the bullet”

意思： 咬牙硬上，硬着头皮做困难但必要的事
场景： 面对痛苦但不得不做的技术决策

A: Refactoring the entire HAL layer will take two sprints.
B: Let’s bite the bullet now before the codebase gets even messier.

引用：「Sometimes you just have to bite the bullet.」— BBC News, 2021

4. “the tip of the iceberg”

意思： 冰山一角，只是问题的表面
场景： 当初步调查暴露更深层问题时

A: We found three memory leaks in the parser module.
B: That’s just the tip of the iceberg — wait until you profile the network stack.

引用：「That was just the tip of the iceberg.」— Titanic, 1997

5. “get the ball rolling”

意思： 开始行动，启动进程
场景： 推动项目从计划进入执行阶段

A: When do we start porting to the new MCU?
B: I’ve already set up the toolchain — let’s get the ball rolling with the GPIO driver.

引用：「It’s time to get the ball rolling.」— The New York Times, 2020
